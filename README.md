# Booze-Bazaar-Online-Alcohol-Store
An online alcohol store, where you can indulge in your favorite spirits from the comfort of your own home. Our website offers a user-friendly interface that allows customers to create an account and easily browse and purchase all kinds of alcohol. 


HOW TO RUN THE FILE:  run 'Index.html'. Thats it.

1. Main page -> has the following
    1. Navbar -> a logo (mp4 attachment.) Can be used to redirect the user to the homepage from any page of the website
    2. Search bar -> use to search for items. Because of the time constrain, it is only able to search for items found in the dummy database. User is able to search for beer, rum, whiskey, wine, vodka, coolers.
                -> users can search for an item with a word that is found in the name or the description of the item found in the dummy database. The search is case sensitive, and will be improved with time
                -> If no item is found, the search page is empty but a back arrow can be used to go back to the main page or the user may use the search bar again to look for an item that is in the database.


    3. Navigations like Heart, Cart and Profile, take the user to the respective pages. Favorites and Cart page is fully implemented. User can add and remove items from both favorite and cart. User is able to increase or decrease quantity of item in the cart and dynamically see the change in the total cost.


    4. The login page is work under progress due to the time constraint
    5. Cards hold links to different pages. Every page takes the user to the listing of items for sale.

    6. The carousel can be used to showcase items on sale, new items in the market, offers on special occasions and discount news for users. The carausel slides are automated to change. But can also be manually changed using the arrows.


-> the Background is a wooden floor to symbolise the old  and rich alcohol.

-> The user may confirm their products in the cart page. After which they can proceed to check out page. Add their information and check out. Finally an order confirmation page opens up for the user to confirm their details. It also has a back to home page button.
